### Requirements

* **Animation:** CSS transform–based, **continuous** loop using duplicated tracks (no snapback). **No timers** (`setInterval`/`requestAnimationFrame`) for movement.
* **Mobile-first layout:**

  * Mobile: single marquee row, full width, 56–72px tall logos.
  * md+: same row with increased gap; optional second “lane” disabled for now.
* **Sizing/visuals:** logos wrapped in fixed-height container (`h-12 sm:h-14`), auto width; subtle grayscale + reduced opacity; on hover (md+) lift grayscale and increase opacity.
* **A11y:** animated row `aria-hidden="true"`. Provide SR-only unordered list of company names below the marquee.
* **Reduced motion:** when `prefers-reduced-motion` is on, render a static grid of logos (no animation).
* **Performance:** SVGs preferred; otherwise provide `width`/`height` attributes on rasters to prevent CLS; `loading="lazy" decoding="async"`.

### 2) Marquee component

```tsx
// client/src/components/animation/InfiniteHorizontalMarquee.tsx
import * as React from "react";

type Logo = { src: string; alt: string; width?: number; height?: number };

export default function InfiniteHorizontalMarquee({
  items,
  speedSeconds = 40,
  className = "",
}: {
  items: Logo[];
  speedSeconds?: number;
  className?: string;
}) {
  const prefersReduce =
    typeof window !== "undefined" &&
    window.matchMedia?.("(prefers-reduced-motion: reduce)").matches;

  // Static fallback
  if (prefersReduce) {
    return (
      <ul className={`grid grid-cols-3 sm:grid-cols-4 md:grid-cols-6 gap-6 ${className}`}>
        {items.map((l, i) => (
          <li key={i} className="flex items-center justify-center h-12 sm:h-14 opacity-80">
            <img
              src={l.src}
              alt={l.alt}
              loading="lazy"
              decoding="async"
              width={l.width}
              height={l.height}
              className="max-h-full w-auto object-contain"
            />
          </li>
        ))}
      </ul>
    );
  }

  // Animated path: duplicate items for seamless 200% track
  const track = [...items, ...items];

  return (
    <div className={`group relative overflow-hidden ${className}`}>
      <div
        aria-hidden="true"
        className="flex animate-marqueeX gap-10 sm:gap-12 md:gap-16 will-change-transform"
        style={{ animationDuration: `${speedSeconds}s` }}
      >
        {track.map((l, i) => (
          <div key={i} className="flex items-center justify-center h-12 sm:h-14 opacity-70 grayscale transition
                                  group-hover:opacity-90 group-hover:grayscale-0">
            <img
              src={l.src}
              alt="" // hidden in animated path; SR-only list provided elsewhere
              loading="lazy"
              decoding="async"
              width={l.width}
              height={l.height}
              className="max-h-full w-auto object-contain"
            />
          </div>
        ))}
      </div>
    </div>
  );
}
```

### What NOT to do

* Don’t animate with JS timers.
* Don’t place the marquee outside the container or with absolute positioning.
* Don’t ship retina-huge rasters; prefer SVG or optimized PNG with explicit `width/height`.

### QA (acceptance)

* Smooth, continuous loop with **no snap** at the seam.
* Mobile: single lane fits the container; logos ~48–56px tall; no horizontal scrollbar.
* Hover (desktop): logos de-grayscale and brighten; animation **pauses** if you want—optional: add `group-hover:[animation-play-state:paused]` to the animated `div` if pausing is desired.
* Reduced motion: static grid visible; screen readers announce the SR-only list once.
* CLS remains near zero (explicit `width/height` on rasters or SVGs).
