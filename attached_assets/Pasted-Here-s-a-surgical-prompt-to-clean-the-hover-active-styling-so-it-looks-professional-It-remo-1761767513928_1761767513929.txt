## Replace current card wrapper styles with this spec

### Card base (default state)

* Container uses a `group` so child elements can react on hover/focus.
* **Base classes** on the clickable card link (`<Link …>`):

  ```
  block h-full rounded-2xl border border-slate-200 bg-white/95 
  shadow-sm backdrop-blur-[0] 
  transition-[transform,box-shadow,background-color,border-color] duration-150 ease-out
  focus:outline-none focus-visible:ring-2 focus-visible:ring-[#0B5FFF] focus-visible:ring-offset-2
  hover:border-slate-300
  ```
* Inner layout (icon + texts) wrapper:

  ```
  flex h-full flex-col justify-between p-4 sm:p-5
  ```

### Icon + text

* Icon wrapper:

  ```
  text-slate-400 transition-colors duration-150
  group-hover:text-[#0B5FFF] group-focus-visible:text-[#0B5FFF]
  ```
* Title:

  ```
  mt-3 text-[clamp(0.95rem,2vw,1.05rem)] font-medium leading-tight 
  text-slate-900 line-clamp-1
  ```
* Subline:

  ```
  mt-1 text-xs text-slate-500 line-clamp-1
  ```

### Hover / Active / Focus styles (professional)

* **Hover (pointer devices)**:

  * Slight lift and sharper shadow:

    ```
    group-hover:shadow-md group-hover:-translate-y-0.5
    ```
  * Background tint only if the card is on a light surface:

    ```
    group-hover:bg-white
    ```
* **Active (mouse down / touch)**:

  * Micro compress for click feedback:

    ```
    active:translate-y-0
    ```
* **Focus (keyboard)**:

  * Keep the blue focus ring from base classes; **no** extra glow.

### Selected/Active page indicator (optional polish)

* If you mark the current page’s cards as “active”, apply:

  ```
  data-active="true":ring-1 data-active:ring-[#0B5FFF]/30
  ```

  Don’t add glow; just a faint ring.

### Remove these (explicit)

* Any `before:` gradient border or shimmer effects.
* Any `scale-[1.02+]` or big movement. Keep to `-translate-y-0.5` max.
* Any `bg-gradient-to-*` on cards.
* Any `drop-shadow-2xl` or heavy shadows.
