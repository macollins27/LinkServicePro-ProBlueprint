You can paste those full objects I wrote above in order.

2. Make sure all `category` and `tags` values are strings. Your filter bar pulls unique categories + tags, so these posts become the initial taxonomy for filters.

3. Replit will use `excerpt` for grid cards, `content` for article body, and `title` for SEO `<Helmet>`.

4. All dates are in ISO so we can run `formatDate()` in the helper.

5. You can update `author.role` later for authority — “15 yrs HVAC ops lead,” etc. That helps conversion. Homeowners don’t care. Contractors do.
