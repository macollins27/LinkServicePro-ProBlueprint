## 3) Slider implementation (no new libs)

**DOM structure**

```tsx
<section aria-label="Industry selection">
  <div className="mx-auto max-w-7xl px-4 sm:px-6 lg:px-8 py-12 sm:py-16">
    <div className="text-center mb-6 sm:mb-8">
      <h2 className="text-2xl sm:text-3xl font-semibold">Built for Your Trade</h2>
      <p className="text-slate-500 mt-2">Workflows tailored to your industry</p>
    </div>

    <div className="relative">
      {/* Prev button md+ */}
      {/* Scroll container */}
      <div
        id="industry-slider"
        className="overflow-x-auto no-scrollbar snap-x snap-mandatory scroll-smooth -mx-4 px-4"
        aria-roledescription="carousel"
        aria-label="Industries"
      >
        <div className="flex items-stretch gap-4">
          {/* CLONED last page */}
          {/* REAL pages ... */}
          {/* CLONED first page */}
        </div>
      </div>
      {/* Next button md+ */}
    </div>

    {/* Dots */}
  </div>
</section>
```

**Page width**

* Each **page** is `min-w-full snap-start` so one page fills the viewport width.
* Compute **itemsPerPage** from breakpoint:
  `base=4`, `sm=6`, `md+=8`.

**Paging logic**

* Split `INDUSTRIES` into arrays of length `itemsPerPage`.
* Create `pages = [ ...chunks ]`.
* Add `clonedHead = pages[0]`, `clonedTail = pages[pages.length-1]`.
* Render order: `[clonedTail, ...pages, clonedHead]`.

**State + effects**

* `const [pageIndex, setPageIndex] = useState(0); // 0..pages.length-1`
* On mount and on resize, recompute `itemsPerPage` and re-chunk pages.
* `onScroll` of slider container: compute current index by `Math.round(scrollLeft / container.clientWidth)`; translate to real page index ignoring clones.
* **Loop jump**: if index === 0 → jump to last real page; if index === pages.length+1 → jump to first real page (set `scrollLeft` immediately without smooth behavior).

**Prev/Next handlers**

* `scrollBy({ left: dir * container.clientWidth, behavior: 'smooth' })`
* Buttons visible with `className="hidden md:flex ..."`; include `aria-controls="industry-slider"` and `aria-label="Previous industries"` / `"Next industries"`.

**Dots**

* Count = `pages.length`.
* Clicking a dot scrolls to `container.clientWidth * (realIndex + 1)` (because of leading clone).

**Keyboard**

* Add `onKeyDown` to the scroll container: Left/Right arrow scroll by one page.
